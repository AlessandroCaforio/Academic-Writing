# Quality Gates — Advisory Scoring

Quality gates are **advisory only**. They NEVER block work or require minimum scores. They provide structured feedback to help improve the thesis.

## Scoring Dimensions (1-10 each)

| Dimension | Description |
|-----------|-------------|
| **Clarity** | Writing is precise, unambiguous, and flows logically |
| **Rigor** | Claims are supported, methods are sound, limitations acknowledged |
| **Consistency** | Notation, terminology, and style are uniform across chapters |
| **Completeness** | All necessary components are present (labels, refs, tables, notes) |
| **Presentation** | LaTeX formatting, table/figure quality, professional appearance |

## Reporting Format

When providing a quality assessment, use this format:

```
Quality Assessment: [scope]
├── Clarity:       X/10 — [one-line note]
├── Rigor:         X/10 — [one-line note]
├── Consistency:   X/10 — [one-line note]
├── Completeness:  X/10 — [one-line note]
└── Presentation:  X/10 — [one-line note]
Overall: X.X/10
```

## Rules

- Scores are informational — they help the user prioritize improvements
- A score below 5 in any dimension should include a specific suggestion
- Never refuse to perform an action based on a quality score
- Quality assessment is optional unless explicitly requested or triggered by a skill
